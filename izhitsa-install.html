Установить на телефон
📱 iPhone
🤖 Android
1
Открой нужный модуль (Магазин, Мастерская или Администратор)
2
Нажми кнопку «Поделиться» внизу Safari (квадрат со стрелкой ↑)
3
Прокрути вниз и выбери «На экран "Домой"»
4
Нажми «Добавить» — иконка появится на рабочем столе
Важно: открывай только через Safari. Chrome на iPhone не поддерживает установку PWA.
1
Открой нужный модуль в браузере Chrome
2
Chrome сам покажет баннер «Установить приложение» — нажми его
3
Или: меню Chrome (⋮) → «Добавить на главный экран»
4
Нажми «Установить» — иконка появится как обычное приложение
Работает офлайн: после первой установки приложение работает без интернета. Данные синхронизируются автоматически при появлении сети.
